- 删除Modal  DeleteModal

- API

|字段                    |说明                          |类型              |默认值     
|---------------------- |------------------------------|------------------|--------------
| showDelModal          | 是否显示                      |boolean            |false
| closeDelModal         | 取消按钮                      |function           |无返回参数
| delData               | 确认按钮                      |function           |无返回参数
| content               | 提示文字                      |string             |确认删除吗？
